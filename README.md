# Weather App

A simple weather app that lets you search for any city and view the current weather along with a 7-day forecast.

## Features

* Search weather by city
* Current weather conditions
* Temperature in **°C** and **°F**
* 7-day forecast
* Weather icons
* Press **Enter** or click the search button to search

## Built With

* HTML
* CSS
* JavaScript
* Visual Crossing Weather API

## Usage

1. Enter a city name.
2. Press **Enter** or click the search button.
3. View the current weather and 7-day forecast.
